Minesweeper

How to Play:

States of cells:

  • Covered: The cell which is not clicked yet.
  • Uncovered: You uncover a covered cell by left-clicking on it.
  • Flagged: Flag a covered cell by right-clicking if you think there is a mine under it.

Uncovering cells

  1. Each uncovered cell shows the number of mines present in the cells surrounding it
  2. The first cell you uncover will not contain any mine
  3. Win by flagging all the cells which contain mine
